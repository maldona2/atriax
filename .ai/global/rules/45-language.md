# Language

## UI Text

- All user-facing text rendered in the browser **must be in Spanish**: labels, placeholders, button text, error messages, success messages, tooltips, headings, descriptions, and any other copy visible to the end user.
- This applies to hardcoded strings in JSX/TSX, default values shown to users, and any string literal that ends up displayed on screen.

## Code

- All code must be written in English: variable names, function names, component names, type names, comments, file names, and Git commit messages.
- Do not mix Spanish into identifiers, comments, or any non-display string (e.g. console.log, API payloads, internal keys).

## Summary

| Context | Language |
|---------|----------|
| UI labels, messages, copy | **Spanish** |
| Code identifiers, comments | **English** |
| API field names / DB columns | **English** |
| Git commits | **English** |
